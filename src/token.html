Token
ที่ใช้ได้จริง

Token ของ G2G ใช้ ซื้อขายเวลา (ที่ปรึกษา · AI hours · บริการ) และ ซื้อขายสินค้า (ผลไม้ อัญมณี อาหารแปรรูป) · ระบบ closed-loop · ไม่ใช่ e-Money

หลักการ · How it works

Token ทำงานอย่างไร ในเครือ G2G

1

เติม Token ผ่านบริการ G2G

เติมผ่านระบบสมัคร Platform · ซื้อจากร้านในเครือ · ทำ task ในระบบ · แลกจาก loyalty

2

ใช้ซื้อเวลา

ที่ปรึกษา AI Hour · ที่ปรึกษามนุษย์ (กฎหมาย, แพทย์, การเกษตร) · ใช้ premium features

3

ใช้ซื้อสินค้า

ผลไม้ Imperial Fruitia · อัญมณี Le Phaya · อาหารแปรรูป Heaven/Thaidelicious · สุรา DD

4

โอน-แลก ระหว่างผู้ใช้

โอน Token ระหว่างบัญชี · ใช้แทนเงินสดในระบบนิเวศ · กระตุ้น loyalty

หมายเหตุกฎหมาย

G2G Token เป็น closed-loop token ใช้ได้ภายในระบบนิเวศของ G2G เท่านั้น · ไม่ใช่ e-Money และไม่ใช่ cryptocurrency · ไม่ต้องขอใบอนุญาตจากธนาคารแห่งประเทศไทย · ปฏิบัติตาม พ.ร.บ. ระบบการชำระเงิน พ.ศ. 2560
